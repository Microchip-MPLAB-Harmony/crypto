The following settings are recommended for building sam_l11_xplained_pro_hw.
Refer to the notes in sam_l11_secure_init.c for details.
 *  AS = 65000  -- almost all of SAML11 Flash memory
 *  RS = 16300  -- almost all of SAML11 SRAM memory
 *  ANSC = 1024 -- modest non-secure  callable
 *  HEAP = 1000
