5.3.1.1 st_Crypto_Aead_AesCcm_ctx

Description

This structure is used to define context for AES-CCM algorithms.

Structure Elements Data TypeElement NameDescription
uint32_tcryptoSessionIDIt defines the session ID, must be more than zero
crypto_HandlerType_EaeadHandlerType_enEnum for crypto operation handler i.e., SW, HW
uint8_t*ptr_keyKey for the AES-CCM cipher operation
uint32_taeadKeySizeKey length for the AES-CCM algorithm
uint8_tarr_aeadDataCtx Array for the context variable data storage
