Running the Demonstration

sam_e70_xplained_ultra.X and sam_e70_xplained_ultra_sw.X 

 

This demonstration exercises various encryption, decryption, hashing and random number functions.

  1. First connect the board to the PC as described in the 'Configuring the Hardware' section.
  2. Configure a terminal application (ex. Tera Term) to access the newly attached serial port. The parameters to use:
    • 115,200 bps
    • 8 data bits
    • no parity
    • 1 stop bit
    • no flow control
3. Compile and program the target device. 

4. Observe the output from the serial console, it should display as follows. 

 

 

sam_e70_xplained_ultra_rtos.X 

 

This demonstration exercises various encryption, decryption, hashing 

and random number functions using freeRTOS.

  1. First connect the board to the PC as described in the <link Con
  2. Compile the target device.
  3. In the MPLABX IDE Projects pane, expand the project.
  4. Under Source Files, double click on app.c to open the file.
  5. Set breakpoints on the lines with the following text:
    • 'One or more tests failed.'
    • 'All tests passed'
These are lines 2101 and 2103 in the code released with the MPLAB Harmony Crypto Library version 3.1.0. As illustrated following. 

  1. Click the Debug Main Project icon to execute the tests.
  2. Note where the application stops. If the app stops on line 2101, then one or more tests failed. If the app stops on line 2103, then all tests passed
  3. Click the Finish Debugger Session to close the application.
